# DareDash UI Adapter

DareDash is the default MAWA UI adapter.

Operational docs are loaded from source whenever a stage requires UI-specific decisions and the docs are not already available in the current session. No committed local copy is required.

Source:

```text
https://raw.githubusercontent.com/pisandelli/daredash/main/llms.txt
```

This adapter provides rules, setup guidance, checklist, and design handoff constraints for Nuxt 4 + DareDash projects.
