# Architecture Principles

MAWA optimizes for predictable AI-assisted delivery, not maximum theoretical flexibility.

## Principles

- Nuxt 4-first core.
- Adapter-based UI and design choices.
- Specs before code.
- Stable artifact paths.
- Human-confirmed setup.
- Store-first business orchestration.
- Explicit context loading.
- Incremental implementation.
- Review before expansion.
